# 🚀 Jai Kishore | Portfolio Website

A modern, responsive, and recruiter-focused personal portfolio website built to showcase my professional journey, technical skills, projects, leadership experience, and achievements.

Designed with a clean **dark theme**, this portfolio serves as both my online resume and personal brand, helping recruiters and hiring managers quickly understand my background and capabilities.
